# 📊 Hacker News Data Analytics Project

## 🚀 Overview
This project fetches trending posts from Hacker News API, cleans the data, performs analysis, and visualizes insights.

## 📌 Features
- Fetch top 20 trending posts
- Clean & preprocess data using Pandas
- Perform analysis using NumPy
- Visualize top posts using Matplotlib

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Hacker News API
